TinyGenie. Submit text

Trained from scratch · 4GB laptop GPU · zero scraped social data

TinyGenie is a small language model being built line-by-line on a laptop GPU — English, Chinese, and everything that happens when people mix the two. Right now it only knows what it's been fed, and most of that is formal text off the open web. Type something below and watch it choke on it.

Models

TinyGenie: MultiLangSG

Building in progress

This model supports English, Chinese and Singlish. This is our first ever model.

WARNING: This model is trained to swap within English and Chinese, and use Singlish at the same time. An alternative is MultiLang.

This model is currently accepting Google Forms responses.


TinyGenie: MultiLang

In queue

This model supports English, Chinese and Malay.

WARNING: This model is trained to use only 1 language at a time, based on the input. If you prefer language swapping, try out MultiLangSG (for slangs), MultiLangSW (all languages but swaps), or MultiLangSWT (all languages + Tamil)

This model is currently NOT accepting Google Forms responses.
